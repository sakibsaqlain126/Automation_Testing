# Automation Testing Project

This repository contains automated test scripts developed using **Playwright** to validate key functionalities of a web application. The project focuses on automating user flows such as login, article summary generation, podcast summary generation, and public link summary generation.

## Project Overview

The main objective of this project is to practice and demonstrate web automation testing using Playwright. The test scripts interact with web elements, perform user actions, and verify application behavior across different features.

## Features Automated

* User login with email and password
* Article summary tool automation
* Podcast summary tool automation
* Public link summary tool automation
* Form input automation
* Button click and navigation flow testing
* Waiting for selectors and visible elements

## Tools & Technologies

* Playwright
* JavaScript
* Node.js
* GitHub
* VS Code


## Learning Outcomes

* Learned how to automate browser actions using Playwright.
* Practiced writing test scripts using JavaScript.
* Gained experience with selectors, locators, test IDs, and form automation.
* Automated real user journeys such as login and content summary generation.
* Improved understanding of software testing, debugging, and test execution workflows.
